Introduction

On top of the extensive Core library, Facet provides a large collection of addational modules and classes.

Usage

Use the More library like ou would any other 3rd party library. The only difference between Facets More library and other thrid party libraries is the lack of a enclosing @Facets::@ namespace. This is becuase the libraries provided by Facets are considerd fairly low-level. (Albeit there are a few exceptions.)
